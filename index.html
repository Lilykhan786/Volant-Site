VOLANT

An open source programming language

What is Volant?

Volant is a general-purpose data-oriented high-level programming language with a focus on speed, power, asynchronicity, and concurrency. Volant is designed with simplicity in mind, allowing both low-level and high-level features with simple and easy to use syntax. Volant is transpiled to C with no or a very thin layer of abstraction over the code you write, which makes it as fast as C in almost all cases.

Volant is meant for good programmers. While volant does have a strong focus on simplicity, it does not focus on an easy learning curve, memory safety, or anything else that can restrict its power.

Why Volant

Mixed level

Volant supports both high level and low level programming. It is a great blend of low and high level features, because why pick one of them when you have both? you can choose to use all the low level features of C or the modern asynchronous programming. It's all about what you want to do and how you want to do it.

General purpose

Volant was not designed with any one goal in mind, you can use it for anything you want to do. Make a dev tool, code a web backend, write a game, go hardware mode and do some embedded systems programming or build and train ML models: your call!

Speed

Volant is a transpiled language. It is transpiled to C with no or a very thin layer of abstraction over the code written. This makes it as fast as C in almost all cases. All of this, while giving access to higher level data structures like vectors and modern features like asynchronicity and concurrency
